ANTES DE PUBLICAR
-----------------
1. Supabase → SQL Editor → cole sql/001_rh_justificativa_faltas.sql → Run
2. Cadastre funcionários ativos:
   - Opção A: sql/002_cadastro_colaboradores_ativos.sql (edite os nomes)
   - Opção B: planilha colaboradores_rh.xlsx + python gerar_sql_colaboradores_rh.py
3. GitHub → lubrificacaomaquinassv-cloud/sigcf-rh
4. Streamlit Cloud → Main file: rh_app.py
5. Secrets: SUPABASE_URL, SUPABASE_KEY, APP_PIN

CADASTRO DE COLABORADORES
-------------------------

Mesma tabela usada por Apontamento, Oficina e Gestor — uma fonte única.
